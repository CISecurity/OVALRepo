Module Hierarchy

  • build_all_oval_definitions: Builds all OVAL definitons in the repository.
  • build_oval_definitions_file: Builds a schema-valid OVAL definitions file.
  • bulk_generate_schematron: Generates all schematron files in parallel.
  • find_affected: Identify changes in current working directory as compared to a remote authoritative copy of the repo and identify all the elements affected by those changes.
  • get_min_oval_version: Determines the minimum OVAL schema version for a given definition fragment
  • get_related_elements: Get OVAL elements that are related to one or more elements.
  • get_repository_stats: Get repository content statistics.
  • lib_git: Library for retreiving lists of updated files using Git.
  • lib_oval: Library to simplify working with the OVAL XML structure
  • lib_repo: Library for accessing files in this repository.
  • lib_search: Library for indexing and searching OVAL elements in the repository.
  • lib_xml: Library for parsing, creating and validating XML files.
  • oval_decomposition: Splits the OVAL document into its component parts and stores them in the appropriate places in the repository.
  • submission_qa: Steps through all of the QA tasks needed for a submission and updates the local repository as necessary on success.
  • validate_oval_definitons_files: Schema validates one or more OVAL definitions files.
  • web_search_defintions: Powers website OVAL definitions search.
